Description
Returns a Type object that is equal to
     log(x) / log(10)

Type Requirement
This functions uses that definition for the log function which is automatically included (in the CppAD namespace) for the following base types : float, double, complex<float>, and complex<double>. If log10 is used with any other base type, log must be a defined for that base type.

Example
The file Log10.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
Input File: CppAD\local\Log10.h
