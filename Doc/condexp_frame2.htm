Description
Returns a object result that is given by
     if( exp1 > Base(0) )
          result = exp2
     else result = exp3
where Type is float or double or AD<Base> where Base is a valid type for conditional expressions. If Type is AD<Base>, conditional expressions are taped and transferred to the corresponding ADFun object f. The choice of expression values is made each time f.Forward is used to evaluate the zero order Taylor coefficients. (See Discrete for another type of taped evaluation).

Example
The file CondExp.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
