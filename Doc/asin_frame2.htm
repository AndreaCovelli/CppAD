Description
Returns an AD<Base> object that is equal to
     asin(x)
where asin has the same interpretation as for the corresponding base type .

Base Type Requirement
A definition for the asin function is automatically included (in the CppAD namespace) for the following base types : float and double. If the asin function is used with any other base type, it must be a defined for that base type.

Derivative Assumption
It is assumed that  \[
\begin{array}{lcr}
     \D{[ {\rm asin} (x) ]}{x} & = & (1 - x * x)^{-1/2}
\end{array}
\] 

Example
The file Asin.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
