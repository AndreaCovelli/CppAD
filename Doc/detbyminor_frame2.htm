Inclusion
The template function DetByMinor is defined in the CppAD namespace by including the file Example/DetByMinor.h. It is only intended for example and testing purposes, so it is not automatically included by CppAD.h .

Constructor

constructs the object Det which can be used for evaluating the determinant of n by n matrices using expansion by minors.

Evaluation
The syntax

returns the determinant of A using expansion by minors.
