Description
Returns an AD<Base> object that is equal to
     log(x)
where log has the same interpretation as for the corresponding base type .

Base Type Requirement
A definition for the log function is automatically included (in the CppAD namespace) for the following base types : float, double, complex<float>, and complex<double>. If the log function is used with any other base type, it must be a defined for that base type.

Derivative Assumption
It is assumed that  \[
\begin{array}{lcr}
     \D{[ \log (x) ]}{x} & = & \frac{1}{x}
\end{array}
\] 

Example
The file Log.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
