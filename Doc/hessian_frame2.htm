This:  section, frame
Description
Given an ADFun<Base> object  F : B^n \rightarrow B^m , this routine returns the Hessian of  F_i at the point x; i.e.,  \[
     H = F_i^{(2)} (x)
\] 

F
The object F has prototype
     ADFun<Base> F

VectorBase
The type VectorBase must be a SimpleVector class with elements of type Base . The routine CheckSimpleVector will generate an error message if this is not the case.

x
The vector x has prototype
     const VectorBase &x
It must have length n and specifies that point at which to evaluate the Hessian.

i
The index i has prototype
     size_t i
It must be between zero and  m-1 and specifies the component of F for which we are evaluating the Hessian.

H
The vector H has prototype
     VectorBase H
It must have length n * n. After the assignment to H, for  j = 0 , \ldots , n - 1  and  \ell = 0 , \ldots , n - 1  \[
     H[ j * m + \ell ] = \DD{ F_i }{ x_j }{ x_\ell } ( x )
\] 

Forward
The object F stores information related to previous calls to F.Forward. After this operation, the previous calls to Forward are undefined with the exception of the zero order call which has the form
     F.Forward(0, x)
.

Examples
The routine Hessian is both an example and a test. It returns true, if it succeeds and false otherwise.
