Computed Assignment Operators

Definition
An operator Op= is a base type computed assignment operator if for each pair of Base objects x and y,
     x Op= y
is defined has the same effect at
     x = x Op y
For example, if Base is double, += is a base type computed assignment operator, but == is not.
