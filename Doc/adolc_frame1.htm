Routines for Comparing CppAD with ADOLC

Description
These examples and speed tests are used to compare ADOLC with CppAD. They are located in the directory CppAD/ADOLC. Note that they can only be run under Unix (or Cygwin on Microsoft windows).

ADOLC Install Directory
The following text appears in the file CppAD/ADOLC/Makefile.
     ADOLC = d:/ADOLC18
 Before you compile and run these tests you will have to replace the text d:/ADOLC18 with directory where ADOLC is installed on your machine.

Example
In a unix operating system (or in a Cygwin shell on an Microsoft system), you can compile link and run this list examples and results comparison between ADOLC and CppAD using the following commands:
     cd CppAD/ADOLC
     make  Example
     Example
Speed
In a unix operating system (or in a Cygwin shell on an Microsoft system), you can compile link and run the list of speed comparisons between CppAD and ADOLC using the following commands:
     cd CppAD/ADOLC
     make  Speed
     Speed
