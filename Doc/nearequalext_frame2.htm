This:  section, frame
Description
This is an extension of the NearEqual routine to more cases. It returns true if  \[
     x = y
     \; {\rm or} \;
     | x - y | \leq a
     \; {\rm or} \;
     \frac{ | x - y | } { |x| + |y| } \leq r
\]  Otherwise, it return false. Note that a is an absolute error bound and r is a relative error bound.
