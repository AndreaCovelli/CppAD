Sine and Cosine Functions Reverse Mode Theory
We use the reverse theory standard math function definition for the functions  H and  G . In addition, we use  s and  c for the p-th order Taylor coefficient row vectors corresponding to  \sin [ X(t) ] and  \cos [ X(t) ] . For the sine and cosine function cases,  \[
     z^{(j)} = ( s^{(j)} , c^{(j)} )
\]  in the definition for  G and  H . The forward mode formulas for the sine and cosine functions are  \[
\begin{array}{rcl}
     s^{(j)}  & = & \sin ( x^{(0)} ) \\
     c^{(j)}  & = & \cos ( x^{(0)} )
\end{array}
\]  for the case  j = 0 , and for  j > 0 ,  \[
\begin{array}{rcl}
s^{(j)} & = & \frac{1}{j} 
     \sum_{k=1}^{j} k x^{(k)} c^{(j-k)}  \\
c^{(j)} & = & - \frac{1}{j} 
     \sum_{k=1}^{j} k x^{(k)} s^{(j-k)} 
\end{array}
\]  If  j = 0 , we have the relation  \[
\begin{array}{rcl}
\D{H}{ x^{(j)} } & = & 
\D{G}{ x^{(j)} }  
+ \D{G}{ s^{(j)} } c^{(0)}
- \D{G}{ c^{(j)} } s^{(0)}
\end{array}
\]  If  j > 0 , then for  k = 1, \ldots , j-1  \[
\begin{array}{rcl}
\D{H}{ x^{(k)} } & = & 
\D{G}{ x^{(k)} }  
+ \D{G}{ s^{(j)} } \frac{1}{j} k c^{(j-k)}
- \D{G}{ c^{(j)} } \frac{1}{j} k s^{(j-k)}
\\
\D{H}{ s^{(j-k)} } & = & 
\D{G}{ s^{(j-k)} } - \D{G}{ c^{(j)} } k x^{(k)}
\\
\D{H}{ c^{(j-k)} } & = & 
\D{G}{ c^{(j-k)} } + \D{G}{ s^{(j)} } k x^{(k)}
\end{array}
\]
