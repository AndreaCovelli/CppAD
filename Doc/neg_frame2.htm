Description
The operation
     - right
returns the additive inverse of right. The operand right is not changed by this operation. If the operand is a variable, the result is a variable .

Assumptions
If the negation (unary -) operator is used with an AD<Base> object, it must also be defined for the base type Base. In addition, if  f is a Base function ,  \[
     \D{[ - f(x) ]}{x} = - \D{f(x)}{x}
\] 

Example
The file Neg.cpp contains an example and a test of this operation.
