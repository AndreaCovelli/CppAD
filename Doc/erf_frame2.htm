Description
Returns the value of the error function which is defined by  \[
{\rm erf} (x) = \frac{2}{ \sqrt{\pi} } \int_0^x \exp( - t * t ) {\bf d} t
\] 

Sequence of Operations
All of the conditional branches in this implementation use the CondExp function. Hence the sequence of operations does not depend on the value of x; i.e., you do not have to record this function for different argument values.

Example
The file Erf.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
