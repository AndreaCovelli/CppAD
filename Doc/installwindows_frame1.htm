Windows Installation and Testing

Download Current Version
If you are using Cygwin or, MinGW with MSYS , with Unix formatted files, just follow all of the Unix installation steps. Otherwise, download the file current version of the file CppAD.dos.tar.gz

Extraction
Use WinZip or some other program to decompress the file CppAD.dos.tar.gz to CppAD.dos.tar and to extract the individual files from CppAD.dos.tar into the distribution directory
     cppad-yy-mm-dd
All the command sequences below assume that you start in the distribution directory; i.e., the one that is created by this extraction. If you are using Cygwin, or MinGW with MSYS, (with dos file formats) you should follow the Unix installation steps for the rest of the install. Otherwise, continue with the steps below:

User Documentation
The Doc sub-directory of the distribution directory contains user documentation for CppAD. The files Doc/*.htm are in HTML format. If your browser supports XHTML+MATHML format, an easier to read copy is available in the files Doc/*.xml.

Developer Documentation
The Developer sub-directory of the distribution directory contains developer documentation for CppAD.

Getting Started
Using Microsoft Visual C++, open the workspace GetStarted/GetStarted.dsw in Visual Studio and then select Build | Build GetStarted.exe. Then in a Dos box, execute the following commands
     cd GetStarted\Debug
     GetStarted
Extensive Examples and Testing
Using Microsoft Visual C++, open the workspace Example/Example.dsw in Visual Studio. Then select Build | Build Example.exe. Then in a Dos box, execute the following commands
     cd Example\Debug
     Example
Printing During Forward Mode Example and Testing
Using Microsoft Visual C++, open the workspace PrintFor/PrintFor.dsw in Visual Studio. Then select Build | Build PrintFor.exe. Then in a Dos box, execute the following commands
     cd PrintFor\Debug
     PrintFor
More Correctness Testing
Using Microsoft Visual C++, open the workspace TestMore/TestMore.dsw in Visual Studio and then select Build | Build TestMore.exe. Then in a Dos box, execute the following commands
     cd TestMore\Debug
     TestMore
Speed Testing
Using Microsoft Visual C++, open the workspace Speed/Speed.dsw in Visual Studio and then select Build | Build Speed.exe. Then in a Dos box, execute the following commands
     cd Speed\Release
     Speed
