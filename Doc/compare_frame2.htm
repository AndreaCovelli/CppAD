Description
Suppose that either left or right is an AD<Base> object, and the other operand is an int, Base or AD<Base> object. In this case
     left Op right
returns a bool value (true or false) where Op is any of the following operators:
Op True Case
< left is less than right
<= left is less than or equal right
> left is greater than right
>= left is greater than or equal right
== left is equal to right
!= left is not equal to right

Definition
An operator Op is a Base comparison operator if for each pair of Base objects x and y,
     x Op y
is defined and returns a bool value. For example, < is a double comparison operator but + is not a double comparison operator (see Base type binary operator ).

Assumptions
If one of the Op operators listed above is used with an AD<Base> object, it is assumed that the same operator is supported by the base type Base.

Example
The file Compare.cpp contains an example and a test of these operations. It returns true if it succeeds and false otherwise.
