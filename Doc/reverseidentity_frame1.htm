An Important Reverse Mode Identity
The theorem and the proof below is a restatement of the results on page 236 of Evaluating Derivatives .

Theorem
Suppose we are given an differentiable  F : B^n \rightarrow  B^m . We define the functions  X : B \times B^{n \times p} \rightarrow B^n ,  Y : B \times B^{n \times p }\rightarrow B^m , and  y^{(j)} : B \times B^{n \times p }\rightarrow B^m by  \[
\begin{array}{rcl}
     X(t, x)  & = & x^{(0)} + x^{(1)} t + \cdots + x^{(p)} t^p
\\
     Y(t, x)  & = & F [ X(t, x) ]
\\
     y^{(j)} (t, x) & = & \frac{1}{j !} \Dpow{j}{t} Y(t, x)
\end{array}
\]  where  x^{(j)} denotes the j-th column of  x \in B^{n \times p} . It follows that for all  i, j such that  i \leq j \leq p ,  \[
\begin{array}{rcl}
\D{ y^{(j)} }{ x^{(i)} } (0, x) & = & \D{ y^{(j-i)} }{ x^{(0)} } (0, x)
\end{array}
\] 

Proof
If follows from the definitions that  \[
\begin{array}{rclr}
\D{ y^{(j)} }{ x^{(i)} } (t, x)
& = & 
\D{ }{ x^{(i)} } \frac{1}{j ! } \Dpow{j}{t} 
     (F \circ  X) (t, x) 
\\
& = &
\frac{1}{j ! } \Dpow{j}{t} 
\D{ }{ x^{(i)} } 
     (F \circ  X) (t, x) 
\\
& = &
\frac{1}{j ! } \Dpow{j}{t} 
[ t^i ( F^{(1)} \circ X ) (t, x) ] 
& \hspace{2cm} (*)
\end{array}
\]  Using the previous equation (*) we obtain  \[
\D{ y^{(j-i)} }{ x^{(0)} } (t, x)
=
\frac{1}{(j-i) ! } \Dpow{j-i}{t} 
( F^{(1)} \circ X ) (t, x)
\]  Note that  i \leq j and if  i < k \leq j the k-th derivative of  t^i is zero. Starting with equation (*), and using  k to index the number of derivatives of  t^i , we have  \[
\begin{array}{rcl}
\D{ y^{(j)} }{ x^{(i)} } (t, x)
& = &
\frac{1}{j ! } \Dpow{j}{t} 
[ t^i ( F^{(1)} \circ X ) (t, x) ] 
\\
& = &
\frac{1}{j !} \sum_{k=0}^i 
\left( \begin{array}{c} j \\ k \end{array} \right)
\frac{ i ! }{ (i - k) ! } t^{i-k} \; 
\Dpow{j-k}{t} ( F^{(1)} \circ X ) (t, x)
\\
& = &
\frac{1}{j !} \sum_{k=0}^i 
\frac{ j ! }{ k ! \; (j - k) ! }
\frac{ i ! }{ (i - k) ! } t^{i-k} \; 
(j - k) ! \; \D{ y^{(j-k)} }{ x^{(0)} } 
\end{array}
\]  Substituting zero for  t , only the term corresponding to  k = i is non-zero, so we have  \[
\begin{array}{rcl}
\D{ y^{(j)} }{ x^{(i)} } (0, x)
& = &
\frac{ j ! \; i ! \; (j - i) ! }{ j ! \; i ! \; (j - i) ! }
\D{ y^{(j-i)} }{ x^{(0)} } (0, x)
\\
& = &
\D{ y^{(j-i)} }{ x^{(0)} } (0, x)
\end{array}
\]
