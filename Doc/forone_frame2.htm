Description
Given an ADFun<Base> object  F : B^n \rightarrow B^m , this routine computes the partial derivative  \[
\D{F}{ x_j } = \left[ 
     \D{ F_0 }{ x_j } (x) , \cdots , \D{ F_{m-1} }{ x_j } (x) 
\right]
\] 

F
The object F has prototype
     ADFun<Base> F

VectorBase
The type VectorBase must be a SimpleVector class with elements of type Base . The routine CheckSimpleVector will generate an error message if this is not the case.

x
The vector x has prototype
     const VectorBase &x
It must have length n and specifies that point at which to evaluate the partial derivatives.

j
Is the argument has prototype
     size_t j
It is the component for which we are computing the partial derivatives,  j  < n .

dF
The vector dF has prototype
     VectorBase dF
It must have length m. After the assignment to dF, for  i = 0 , \ldots , m - 1   \[.
     dF[ i ] = \D{ F_i }{ x_j } ( x )
\] 

Forward
The object F stores information related to previous calls to F.Forward. After this operation, the previous calls to Forward are undefined with the exception of the zero order call which has the form
     F.Forward(0, x)
.

Examples
The routine ForOne is both an example and a test. It returns true, if it succeeds and false otherwise.
