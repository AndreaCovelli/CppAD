Description
This form of printing values is delayed until Forward mode is called with order equal to zero; i.e., when the value of the independent variables is changed. This is can be helpful for understanding why tape evaluations have trouble, for example, if the result of a tape calculation is the IEEE code for not a number Nan. The text specified by text, followed by the value specified by x, are is printed to std::cout.

The Output section describes the normal printing of values; i.e., printing when the corresponding code is executed.

Example
The program PrintFor.cpp is an example and a test of this operation. The output of this program states the conditions for passing and failing the test.
