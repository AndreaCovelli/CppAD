An Example Function
Given a positive integer  n and a vector  a \in \R^{n+1} we define the function  p : \R \rightarrow \R by  \[
     p(x) = a_0 x^0 + a_1 x^1 + \cdots + a_n x^n
\]  It follows that the derivative of  p(x) is given by  \[
     \D{p}{x} (x) = a_1 + 2 a_2 x + \cdots + n a_n x^{n-1}
\]
