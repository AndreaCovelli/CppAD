Description
Evaluates the p-th order Taylor coefficient for the function  F \circ U defined by  \[
     F \circ X (t) = F[ X(t) ]
\]  where  F : B^n \rightarrow B^m ,  n is the dimension of the domain space for F, and  m is the dimension of the range space for F, and  X : B \rightarrow B^m is defined by  \[
     X(t) = x^{(0)} + x^{(1)} t + \cdots, + x^{(p)} t^p 
\]  Note that for  k = 0 , \ldots , p ,  \[
     x^{(k)} = \frac{1}{k !} X^{(k)} (0) 
\]  is the k-th order Taylor coefficient for  X (see below for how  x^{(k)} \in B^n is specified).

F
The object F has prototype
     ADFun<Base> F

Forward
The function Forward has prototype
template <typename Base>
template <typename VectorBase>
VectorBase ADFun<Base>::Forward(size_t p, const VectorBase &up)

VectorBase
The type VectorBase must be a SimpleVector class with elements of type Base . The routine CheckSimpleVector will generate an error message if this is not the case.

x
For  k = 0 , \ldots , p-1 , the k-th order Taylor coefficient  x^{(k)} \in B^n is the value of xk in the previous call to F.Forward(k, xk). If there is no previous call with  k = 0 ,  x^{(0)} is the value of the independent variables when F was constructed as an ADFun object. The p-th order Taylor coefficient  x^{(p)} is specified by the value of xp in this call to F.Forward(p, xp).

Order
The function call F.Order() returns the highest order of the Taylor coefficients currently stored in F. This must be greater than or equal to  p-1 when F.Forward(p, xp) is called. It will be equal to p after this call.

yp
The return column vector  yp , denoted by  y^{(p)} \in B^m below, has length  m and contains the p-th order Taylor coefficient for  F \circ X ; i.e.,  \[
     y^{(p)} = \frac{1}{p !} * (F \circ X)^{(p)} (0) 
\] 

Zero Order
In the case where p is zero, the return value  y^{(p)} \in B^m is  \[
\begin{array}{rcl}
y^{(p)} & = & y^{(0)} \\
     & = & (F \circ X) ( 0 ) \\
F[ x^{(0)} ]
     & = & y^{(p)}
\end{array}
\] 

First Order
In the case where p is one, the return value  y^{(p)} \in B^m is  \[
\begin{array}{rcl}
y^{(p)} & = & y^{(1)} \\
     & = & (F \circ X)^{(1)} ( 0 ) \\
     & = & F^{(1)} [ X(0) ] *  X^{(1)} (0) \\
     & = & F^{(1)} ( x^{(0)} ) *  x^{(1)}
\end{array}
\]  Note that if  x^{(1)} is the j-th elementary vector  \[
\D{F}{x_j} ( x^{(0)} ) = y^{(p)}
\] 

Second Order
In the case where p is two, the i-th element of the return value  y^{(p)} \in B^m is  \[
\begin{array}{rcl}
y_i^{(p)} 
& = & \frac{1}{2} (F_i \circ X)^{(2)} ( 0 ) 
\\
& = & \frac{1}{2} \left[ F_i^{(1)} [ X(0) ] * X^{(2)} (0) 
  + X^{(1)} (0)^T * F_i^{(2)} [ X(0) ] * X^{(1)} (0) \right]
\\
& = & \frac{1}{2}  \left[
     F_i^{(1)} ( x^{(0)} ) * x^{(2)}
     +
     ( x^{(1)} )^T * F_i^{(2)} ( x^{(0)} ) * x^{(1)}
\right  ]
\end{array}
\]  Note that if  x^{(1)} is the j-th elementary vector and  x^{(2)} is zero,  \[
\begin{array}{rcl}
     \DD{F_i}{x_j}{x_j} ( x^{(0)} ) = 2 y_i^{(p)} 
\end{array}
\]  If  x^{(1)} is the sum of the j-th and l-th elementary vectors and  x^{(2)} is zero,  \[
\begin{array}{rcl}
     y_i^{(p)} 
     & = & \frac{1}{2} \left[
          \DD{F_i}{x_j}{x_j} ( x^{(0)} )
          +
          \DD{F_i}{x_j}{x_\ell} ( x^{(0)} )
          +
          \DD{F_i}{x_\ell}{x_j} ( x^{(0)} )
          +
          \DD{F_i}{x_\ell}{x_\ell} ( x^{(0)} )
     \right]
     \\
     \DD{F_i}{x_\ell}{x_j} ( x^{(0)} )
     & = & 
     y_i^{(p)} 
     -
     \frac{1}{2} \DD{F_i}{x_j}{x_j} ( x^{(0)} )
     -
     \frac{1}{2} \DD{F_i}{x_\ell}{x_\ell} ( x^{(0)} )
\end{array} 
\]

The file Forward.h contains an example and a test of this operation. It returns true if it succeeds and false otherwise.
