Description
Returns an AD<Base> object that is equal to
     sin(x)
where sin has the same interpretation as for the corresponding base type .

Base Type Requirement
A definition for the sin function is automatically included (in the CppAD namespace) for the following base types : float, double, complex<float>, and complex<double>. If the sin function is used with any other base type, it must be a defined for that base type.

Derivative Assumption
It is assumed that  \[
\begin{array}{lcr}
     \D{[ \sin (x) ]}{x} & = & \cos (x)   \\
     \D{[ \cos (x) ]}{x} & = & - \sin (x)
\end{array}
\] 

Example
The file Sin.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
