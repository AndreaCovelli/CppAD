Description
Returns an AD<Base> object specified by  \[
     {\rm pow} (x, y) = x^y
\]  If y is an int, the calculation is performed using repeated multiplications. Otherwise the log of y must be defined. (for example, if y is zero, its log is not defined).

Assumptions
If y is not an int, the derivative assumptions for the log and exp functions must hold.

Standard Base Types
A definition for the pow function is automatically included (in the CppAD namespace) for the following base types : float, double, complex<float>, and complex<double>.

Example
The file PowInt.cpp contains an example and test of this function for the case where y is an int. The file Pow.cpp contains an example and test of this function for the case where y is not an int. Both tests return true if they succeed and false otherwise.
