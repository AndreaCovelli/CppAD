Glossary

Base Function
If Base is a C++ class, a function  f is referred to as a Base function if its domain and range are finite dimensional vector with components in the space represented by Base; i.e.  f : B^n \rightarrow B^m .

Base Type
If x is an AD<Base> object, Base is referred to as the base type for x.

Dependent Variable
An AD<Base> object is a dependent variable if it is a variable and is not an independent variable.

Elementary Vector
The j-th elementary vector  u \in B^m is defined by  \[
u_i = \left\{ \begin{array}{ll}
     1 & {\rm if} \; i = j \\
     0 & {\rm otherwise}
\end{array} \right.
\] 

Independent Variable
The set of independent variables are specified by the Independent function call that starts recording operations on the corresponding tape.

Parameter
All Base objects are always parameters. An AD<Base> object is currently a parameter if it is not currently a variable. All AD<Base> objects are currently parameters when the tape is in the Empty state. An AD<Base> object x is currently a parameter if and only if Parameter(x) is true.

Note that we often drop the word currently and just refer to an AD<Base> object as a variable or parameter.

Tape State
For each class Base, the tape that records AD<Base> operations has two possible states:
     enum TapeState {
          Empty,
          Recording,
     }
 The tape is initially in the Empty state. It is in the recording state after each call to Independent with a AD<Base> argument. It is in the Empty state after each call to ADFun<Base> .

Empty
The tape is empty (hence all AD<Base> objects are parameters). The tape is initially in this state and it enters this state whenever a ADFun object is constructed.

Recording
All of the independent variables have been specified and the tape is currently recording the operations that define the more variables as Base scalar functions of the independent variables. The tape enters this state when a Independent calculation is preformed.

Taylor Coefficient
Suppose  X : B \rightarrow B^n is a is  p times continuously differentiable function in some neighborhood of zero. For  k = 0 , \ldots , p , we use the column vector  x^{(k)} \in B^n for the k-th order Taylor coefficient corresponding to  X which is defined by  \[
     x^{(k)} = \frac{1}{k !} \Dpow{k}{t} X(0)
\]  It follows that  \[
     X(t) = x^{(0)} + x^{(1)} t + \cdots + x^{(p)} t^p  + R(t)
\] where the remainder  R(t) divided by  t^p converges to zero and  t goes to zero.

Variable
An AD<Base> object is currently a variable if the corresponding tape is in the Recording state and the current value of an AD<Base> object depends on the value of at least one of the independent variables. An AD<Base> object x is currently a variable if and only if Parameter(x) is currently false. For example, directly after the code sequence
     Independent(u);
     AD<double> x = u[0];
the AD<double> object x is currently a variable. Directly after the code sequence
     Independent(u);
     AD<double> x = u[0];
     x = 5;
x is currently not a variable (it is currently a parameter ).

Note that we often drop the word currently and just refer to an AD<Base> object as a variable or parameter.
