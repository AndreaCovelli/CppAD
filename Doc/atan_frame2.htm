Description
Returns an AD<Base> object that is equal to
     atan(x)
where atan has the same interpretation as for the corresponding base type .

Base Type Requirement
A definition for the atan function is automatically included (in the CppAD namespace) for the following float and double base types . If the atan function is used with any other base type, it must be a defined for that base type.

Derivative Assumption
It is assumed that  \[
\begin{array}{lcr}
     \D{[ {\rm atan} (x) ]}{x} & = & \frac{1}{1 + x^2}  
\end{array}
\] 

Example
The file Atan.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
