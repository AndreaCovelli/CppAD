This:  section, frame
Description
Returns an AD<Base> value  \theta such that  - \pi \leq \theta and  \theta \leq \pi and  \[
\begin{array}{rcl}
     \sin ( \theta )  & = & y / \sqrt{ x^2 + y^2 }  \\
     \cos ( \theta )  & = & x  / \sqrt{ x^2 + y^2 }
\end{array}
\]  The > operator must be defined for objects of type Base; for example, Base cannot be std::complex<double>.

Example
The file Atan2.cpp contains an example and a test of this function. It returns true if it succeeds and false otherwise.
