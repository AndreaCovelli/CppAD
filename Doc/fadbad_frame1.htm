Routines for Comparing CppAD with FADBAD

Description
These examples and speed tests are used to compare FADBAD with CppAD. They are located in the directory CppAD/FADBAD. Note that they can only be run under Unix (or Cygwin on Microsoft windows).

FADBAD Install Directory
The following text appears in the file CppAD/FADBAD/Makefile.
     FADBAD = d:/FADBAD++
 Before you compile and run these tests you will have to replace the text d:/FADBAD++ with directory where FADBAD++ is installed on your machine.

Example
In a unix operating system (or in a Cygwin shell on an Microsoft system), you can compile link and run this list examples and results comparison between FADBAD and CppAD using the following commands:
     cd CppAD/FADBAD
     make  Example
     Example
Speed
In a unix operating system (or in a Cygwin shell on an Microsoft system), you can compile link and run the list of speed comparisons between CppAD and FADBAD using the following commands:
     cd CppAD/FADBAD
     make  Speed
     Speed
