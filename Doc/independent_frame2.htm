This:  section, frame
Description
Calling the Independent starts the recording of AD<Base> operations. This is the first step in defining a CppAD differentiable function object (ADFun object).

VectorADBase
The type VectorADBase must be a SimpleVector class with elements of type AD<Base> . The routine CheckSimpleVector will generate an error message if this is not the case.

u
The vector u has prototype
     VectorADBase &u
The length of u is the number of independent variables (dimension of the domain space for the ADFun object)

Tape State
The operation can only be done while the tape that records AD<Base> operations is in the Empty state . After this operation, the tape will be in the Recording state.

Parameters
Directly before Independent is called, all other AD<Base> objects are parameters ; i.e., they do not dependent on the independent variables. For example, after the code sequence
     v = u;
     Independent(u);
all of the elements of the vector v are parameters. Hence their partial derivatives with respect to each of the elements of u are identically zero.

Variables
An AD<Base> object is a variable if and only if its value depends on the value of one of the independent variables. For example, after the code sequence
     Independent(v);
     v = u;
all of the elements of the vector v are variables. In fact, the partial of v[i] with respect to u[i] is one.
