Routines That Speed Test CppAD

Description
These examples are in the directory CppAD/Speed.

Unix
In a unix operating system (or in a Cygwin shell on an Microsoft system), you can compile link and run the list of examples and verification tests with the following commands:
     cd CppAD/Speed
     make 
     Speed
Windows
In Microsoft Visual Studio.
     File | Open Workspace | CppAD\Speed\Speed.dsw
     Build | Set Active Configuration | Speed - Win32 Release
     Build | Speed.exe
