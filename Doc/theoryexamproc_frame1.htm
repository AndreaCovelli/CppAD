A Procedural Representation of the Example Function
The following is a C++ template function that calculates the value of the example function :
template <class Base>
Base p(unsigned int n, const std::vector<Base> &a, const Base &x)
{    Base  s = a[n];
     unsigned int i;
     Base i;
     for(i = 1; i <= n; i++)
     {    t = s * x;
          s = t + a[n-i];
     }
     return s;
}
 It follows that the return value of the procedure p is equal to the example function value  p(x) (Poly.h contains an implementation of a similar template function.)
