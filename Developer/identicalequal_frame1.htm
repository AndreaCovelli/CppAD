Check if a Value is Identically Equal Another
Syntax inline bool IdenticalEqual(const Type &x, const Type)

Description
If Type is AD<Base> for some Base, x is identically equal y if and only if both x and y are parameters and x.value is identically equal y.value. If Type is not AD<Base> for some Base, x is identically equal y if and only if
     x == y
