Building and Testing a Distribution

Creating Distribution Files
  1. Change into the CppADpackage directory.
  2. Execute the bash shell script ./Build
This will create the distribution files
     CppADpackage/CppAD.dos.tar.gz
     CppADpackage/CppAD.unix.tar.gz
Testing Distribution
  1. Execute the command
         tar -xvzf CppAD.    sys.tar.gz
    where sys is dos or unix depending on your system type.
  2. Execute the command
         cd cppad-    yy-mm-dd
    where %yy%-%mm%-%dd% is the current date.
  3. Execute the command
         configure ADOLC_DIR=    AdolcDir FADBAD_DIR=FadbadDir
    where the paths
             AdolcDir/include/adolc/adouble.h
             FadbadDir/FADBAD++/badiff.h
    are a valid way to reference the adouble.h file in the ADOLC distribution and the badiff.h in the FADBAD++ distribution.
  4. Execute the command
         make
         If no compiler errors occur, the distribution passed this test.
  5. Execute the command
         cd GetStarted
         ./GetStarted
         If the result is
    Poly  computed derivative = 142
    CppAD computed derivative = 142
    then the distribution passed this test.
