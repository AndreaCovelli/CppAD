Description
This function increases the length of the array pointed to by buffer.

length
is the length of the array pointed to by the new buffer.

number
is the number of elements in the buffer that have important values which must be copied to the new buffer.

buffer
On input, buffer points to number elements of information that must be copied from the old buffer to the new one. On output, buffer points to the new buffer. The data is copied to the new buffer using the Type assignment operation.
